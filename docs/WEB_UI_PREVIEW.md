- **主色調：** 藍色 (#4A90E2) - 信任、安全

- **成功色：** 綠色 (#4CAF50) - 確認、通過

## 交互效果

### 按鈕懸停
```
正常: [按鈕]
懸停: [按鈕] ← 微微上浮 + 陰影加深
```

### 卡片點擊
```
正常: ┌─────┐
      │ 卡片 │
      └─────┘

點擊: ┌─────┐ ← 上浮動畫
      │ 卡片 │   邊框變藍
      └─────┘   陰影增強
```

### 折疊區塊
```
展開: ┌─ 標題 ▼ ─────┐
      │ 內容...      │
      └──────────────┘

收起: ┌─ 標題 ▶ ─────┐
```

## 特色功能展示

### 1. 即時驗證
輸入 API Key 後可立即測試，無需保存配置。

### 2. 視覺化配置
複雜的 YAML 配置轉換為直觀的複選框和下拉選單。

### 3. 模板一鍵載入
新手友好，選擇模板即可快速開始使用。

### 4. 配置預覽
保存前可預覽生成的 YAML，確保正確無誤。

### 5. 移動端適配
在平板或手機上也能方便地調整配置。
